# prettier
- Arrow Parens：avoid（当箭头函数仅有一个参数时去掉括号）
- Print Width：80（代码换行长度）
- Semi：false（去掉行尾分号）
- Single Quote：true（使用单引号）
- Tab Width：4（制表符占用空格数）
- Trailing Comma：none（无尾随逗号）
- Use Tabs：false（不使用tab制表符缩进）
- Vue Indent Script And Style：true（vue单文件script、style标签内首行缩进）
# vuter
- 使用0.3以下版本
- 所有Default formatter选项设置为prettier，没有prettier选项则使用默认值
- Tab Size：4（制表符占用空格数）
- Use Tabs：false（不使用tab制表符缩进）
- Script Initial Indent：true
- Style Initial Indent：true
